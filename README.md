# Hoglet DAO Factory

Open-source DAO infrastructure for Move.

Create DAOs powered by ve(3,3) governance on Supra.

🌐 Live Testnet
https://daos.hoglet.xyz

---

## Why?

Hoglet DAO Factory allows any project to launch a fully on-chain DAO using the ve(3,3) governance model.

Instead of building governance from scratch, projects can deploy a DAO with voting, veTokens, proposals, and treasury management out of the box.

Everything is open source.
Everything is on-chain.
Everything is community governed.
